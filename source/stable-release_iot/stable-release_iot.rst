Stable Celadon IoT
##################

Celadon IoT is an Internet of Things (IoT) overlay on top of Celadon. 
Celadon IoT, allows users to use an extensive set of features on Comet Lake(CML), Tiger Lake(TGL), and Elkhart Lake(EHL) IntelÂ® platforms with single Baseline and single Image. 
Celadon IoT offers 100% compliance for Google Compatibility Test Suite (CTS), Vendor Test Suite (VTS), CTS-on-GSI, GMS Test Suite (GTS), and Security Test Suite (STS) on Comet Lake(CML), Tiger Lake(TGL), and Elkhart Lake(EHL). 
Celadon IoT empowers the community to utilize 100% compliance on the supported platforms with continuous support for new features and fixes.
